## ğŸš€ Getting Started

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/KarunasriGunti/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server for better development experience:
   
   **Using Python:**
   ```bash
   python -m http.server 8000
   ```
   
   **Using Node.js (http-server):**
   ```bash
   npx http-server
   ```
   
   Then visit `http://localhost:8000` in your browser

### Deployment to GitHub Pages

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial portfolio commit"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Navigate to **Settings** â†’ **Pages**
   - Under "Source", select **main** branch
   - Click **Save**
   - Your portfolio will be live at: `https://karunasrigunti.github.io/portfolio/`

## ğŸ¨ Customization

### Colors
Edit CSS custom properties in `style.css`:
```css
:root {
  --color-accent-purple: #8b5cf6;
  --color-accent-blue: #3b82f6;
  --color-accent-cyan: #06b6d4;
}
```

### Content
Update your information in `index.html`:
- **Hero Section**: Update name, title, and description
- **About Section**: Modify professional summary
- **Skills Section**: Add/remove skill tags
- **Experience Section**: Update work history
- **Projects Section**: Showcase your projects
- **Contact Section**: Update email, phone, and social links

### Fonts
Change fonts by updating the Google Fonts import in `style.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=YourFont:wght@400;700&display=swap');
```
